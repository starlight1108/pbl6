# 校园二手交易网站用户故事拆分

## 1. 用户管理模块

### 1.1 用户注册
- **故事ID**: US-UM-001
- **故事描述**: 作为新用户，我希望通过校园邮箱注册账号，以便成为平台的合法用户。
- **验收标准**:
  - 成功使用校园邮箱注册
  - 邮箱验证邮件及时送达
  - 密码强度验证通过
  - 注册流程顺畅无错误
- **优先级**: 高

### 1.2 用户登录
- **故事ID**: US-UM-002
- **故事描述**: 作为注册用户，我希望通过邮箱和密码登录系统，以便访问我的账户和使用平台功能。
- **验收标准**:
  - 正确凭证可成功登录
  - 连续5次密码错误后账号锁定30分钟
  - 登录响应时间≤500ms
  - 登录状态保持7天（勾选"记住我"）
- **优先级**: 高

### 1.3 密码找回
- **故事ID**: US-UM-003
- **故事描述**: 作为忘记密码的用户，我希望通过注册邮箱重置密码，以便重新获得账号访问权限。
- **验收标准**:
  - 成功通过邮箱重置密码
  - 重置链接有效期2小时
  - 新密码符合强度要求
- **优先级**: 高

### 1.4 个人信息管理
- **故事ID**: US-UM-004
- **故事描述**: 作为登录用户，我希望查看和修改个人信息，以便保持个人资料的准确性。
- **验收标准**:
  - 成功修改头像、昵称、联系方式等
  - 头像上传支持jpg、png格式，大小不超过2MB
  - 联系方式可设置是否公开
- **优先级**: 中

### 1.5 实名认证
- **故事ID**: US-UM-005
- **故事描述**: 作为卖家，我希望完成实名认证，以便获得发布商品的权限。
- **验收标准**:
  - 成功提交实名认证申请
  - 认证审核在24小时内完成
  - 学生证照片清晰可辨认
- **优先级**: 高

### 1.6 账号注销
- **故事ID**: US-UM-006
- **故事描述**: 作为用户，我希望申请注销账号，以便保护我的数据隐私。
- **验收标准**:
  - 注销后账号无法恢复
  - 注销前需二次确认
  - 用户数据完全清除
- **优先级**: 低
